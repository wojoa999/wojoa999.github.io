<h1>Project: Textbook Finder</h1>
<h2>Overview</h2>
<i>The problem</i>: Each semester students will need to purchase new textbooks and possibly discard old textbooks. The cost of purchasing a new textbook has the potential to cost a couple hundred dollars. Selling your old textbooks back to the bookstore doesn't even compare to the price that you will pay for the textbook. 

<i>The solution</i>: The Textbook Finder application will allow students to find and discard textbooks required for their new course work. Students can sign up to display what textbooks they possess to either sell or trade their textbooks with other students.   

<h2>Approach</h2>
The textbook finder application will involve three user roles which include a user, seller, and an admin. The user  and seller will be able to browse the textbook listings and have the ability to list textbooks available. The admin role will be able to make changes to keep the application up to date with courses that are not offered or the addition of new courses. 

The possible mockup pages include:
<ul>
<li>Landing page</li>
<li>Sign up/ sign in page</li>
<li>Home page</li>
<li>User profile page</li> 
<li>User listed book page</li> 
<li>Seller page</li>
<li>Admin page</li> 
<li>Book listing page</li> 
</ul>

<h2>Use Case Ideas</h2>
The following bullet points may not list all the functions of the application, but will show the basic functionality of the application.

<ul>
<li>The new user goes to the landing page, creates a new profile, and launches the home page.</li>
<li>The user goes to the landing page, logs in, and launches the home page.</li>
<li>The admin goes to the landing page, logs in, and launches the home page.</li>
<li>The user from the home page will be able to make a book listing or browse books.</li> 
<li>The admin from the homepage will be able to ensure that the course listing is up to date.</li> 
</ul>

<h2>Beyond the Basics</h2>
Upon completion of the basic application functions, more advanced features for the application include:
<ul>
<li>Adding a page where users have the potential to share textbooks.</li>
<li>The addition of notifications when a textbook you are looking for is added to the application.</li> 
<li>Create a review page to allow users to review the sellers.</li>
<li>Adding filters to allow users to quickly find the course textbook they are looking for.</li> 
<li>Add the functionality of how long the textbook has been on the market.</li>  
</ul>
